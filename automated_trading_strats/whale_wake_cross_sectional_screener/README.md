# Whale Wake Cross-Sectional Screener

This strategy folder stores the thesis and minimal metadata for the cross-sectional options screener built around Hurst persistence, drift, and ask-vs-model edge ranking.

Suggested local-only subpaths for future automation:

- `runs/` for daily scan outputs
- `logs/` for bot logs
- `cache/` for market data snapshots

Those paths are already covered by the repo-level `.gitignore` rules for `automated_trading_strats/`.
